# SQL tutorial #

## Format ##
A sequence of [Python] modules that demonstrate various aspects of SQL using SQLite databases. The modules are named in the order in which we'll cover them (01_descriptive_name.py, 02_descriptive_name.py, ... N_descriptive_name.py).
